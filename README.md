# Flagship CFO + ERM Dashboard

A polished portfolio-ready Streamlit dashboard that combines:
- quarterly financial position
- annual 990-style indicators
- enterprise risk exposure
- executive recommendations

## Files
- `app.py`
- `quarterly_financial_position.csv`
- `annual_990_snapshot.csv`
- `risk_register.csv`
- `requirements.txt`

## Positioning
Use this as a flagship portfolio sample for:
- ERM and operational risk roles
- nonprofit or higher education finance analytics
- board or executive reporting dashboards
- scenario analysis and decision-support work

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
