# Financial Risk Scenario Engine (2025)

## Overview
This project is a scenario-based financial and enterprise risk model designed for a mission-driven organization with approximately $10M in annual revenue.

It demonstrates how financial data, risk exposure, and scenario modeling can be integrated into a decision-support system for planning, governance, and executive review.

## What This Demonstrates
- Scenario analysis for financial and operational planning
- Integration of ERM and financial performance
- Baseline vs scenario comparison
- Deviation analysis for leadership insight
- Executive-readable financial formatting

## Key Features
- Baseline vs scenario vs deviation comparison
- Preset scenarios for rapid demonstration
- Clean $K / $M formatting
- Synthetic risk register
- Synthetic 990-style financial dataset

## Files
- `app.py` — interactive dashboard application
- `synthetic_risk_register.csv` — simulated risk register data
- `synthetic_990_financials.csv` — simulated financial dataset
- `requirements.txt` — project dependencies

## Example Use Cases
This type of model could support:
